Htown Bars

Htown Bars

Personal Projects

HTOWN Bars is an interactive & intuitive mobile-friendly site for finding your next favorite bar in the Greater Houston Area. This project uses a NoSQL Database to host an inventory of approximately 200 bars in the Houston Area, with more to come. Each bar on the list is rated on a set of 12 unique criteria from Beer selection, to Parking availability, to Decor and so on. The site provides an interactive carousel on the results page which shows the top 10 bars returned based on the input criteria as well as an embedded Google Maps so the user can get a quick overview of where all their selected bars reside. On your next visit to Houston, be sure to use HtownBars to find your next watering hole.
